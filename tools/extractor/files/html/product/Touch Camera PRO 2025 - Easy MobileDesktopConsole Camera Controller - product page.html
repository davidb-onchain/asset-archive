Touch Camera PRO is a really easy to use mobile/desktop/console camera controller with perspective switching! It is used inside other plugins like Home Designer, Runtime Level Editor and Floor Map Designer!


  • Working on desktop, mobile devices and console gamepads!
  • One clean camera prefab, no nested hierarchy
  • Many configuration variables
  • Support translation, rotation around center, rotation around point, Zoom In/Out
  • Support multi fingers "Twist" and "Pinch" around fingers center
  • Support scene bounderies
  • 5 perspective modes are available : 3D Perspective, Top Down Orthographic, Isometric Orthographic, Side Scroll, Free Mode
  • Smooth transition between perspective modes at any time!
  • Full C# included, and commented
  • 20 Demo scenes included
  • Focus on object / Follow object feature
  • Move To (position, [rotation], [distance])
  • Reset Camera to initial position & rotation
  • You can now choose between square and circle boundaries
  • Change ground height easily
  • Free Camera mode (beta)
  • New API methods and documentation
  • New inertia options
  • Now supports Cinemachine!
  • Now supports controllers through the new Input System!
  • New: Now compatible with Unity6
