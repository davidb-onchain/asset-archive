This simple to use package allows you to easily add different camera styles into your project. All styles are customisable in the inspector via sliders and toggles.


The package can be configured for the following camera styles:


  • First person,
  • Third person: Follow,
  • Third person: Freelook,
  • Top-down,
  • Room camera,
  • Multi-Target Camera,
  • Cinematic Camera,
  • Free Fly Camera (Photo Mode).

The camera system works in single player and multiplayer projects! The cameras use their own seperate trigger system to change / activate other views.

Requirements:

  • GgCore Package is required for this package to function. You can purchase GgCore for Free with any account that has previously purchased any Gaskellgames package.
  • Input Event System Package is required for this package to function. You can pick up the Input Event System for Free.

Camera Styles:


  • First person,
  • Third person: Follow,
  • Third person: Freelook,
  • Top-down,
  • Room camera,
  • Multi-Target Camera,
  • Cinematic Camera,
  • Free Fly Camera (Photo Mode).

Camera Features:


  • Camera collisions
  • Camera blending (Cut, Fade & Move to Position)
  • Camera shake (impulse)

Coding Level Requirement:


  • None: You can simply add the camera componants and adjust everything in the inspector
